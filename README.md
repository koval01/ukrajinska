# ukrajinska 🇺🇦

[![PyPI version](https://badge.fury.io/py/ukrajinska.svg)](https://pypi.org/project/ukrajinska/)
[![Python versions](https://img.shields.io/pypi/pyversions/ukrajinska.svg)](https://pypi.org/project/ukrajinska/)

A sleek, bidirectional Ukrainian transliteration library for Python. 
It implements **DSTU 9112:2021 (System A)** – the official national standard for Romanization of Ukrainian.

## Installation

```bash
pip install ukrajinska
```
